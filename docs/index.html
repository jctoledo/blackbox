What Blackbox Does

📡

Real-Time Sensor Fusion

Extended Kalman Filter fuses 200Hz IMU data with 5Hz GPS for smooth, accurate position and velocity estimates.

📍

Position Tracking

Track your vehicle's position on a local coordinate system with automatic GPS warmup and reference point averaging.

🏎

Mode Detection

Automatically classifies driving state: idle, acceleration, braking, or cornering based on sensor data.

📶

Wireless Streaming

Binary telemetry over UDP at 20Hz. Compact 67-byte packets optimized for real-time analysis.

⚡

Zero-Velocity Updates

ZUPT prevents drift when stationary. Accelerometer bias is continuously estimated and corrected.

🛠

Open Source

MIT licensed. Full Rust firmware, Python receivers, and documentation. Build, modify, extend.
